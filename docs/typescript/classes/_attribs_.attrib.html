Class Attrib

Attrib class. An class that represents a semantic attribute. An attribute is data that is attached to a specific type of geometry. The geometric types to which attributes can be attached are: "points", "vertices", "edges", "wires", "faces", and "objs". An instance of this class stores a list of attributes values.

    • Defined in attribs.ts:85

    Get a single attribute value. The data type of the attribute value can be found using the getDataType() method. If getGeomType() returns "points" or "objs", then path should be the entity id (a number). If getGeomType() returns "vertices", "edges", "wires", or "faces", then path should be a path to a topo component (an instance of GeomPath).
