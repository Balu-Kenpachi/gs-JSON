    The purpose of this method consists in creating a Point in the Geometry. The geometry class has two private properties which include points_data. points_data array gathers pointers and cartesian xyz coordinates. The addPoint creates a point of coordinates xyz, and modifies the points_data array by pointing to an already existing xyz value or by creating it if not referenced.
